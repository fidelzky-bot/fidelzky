# Project Images

Place your project screenshots/images here.

## Image Requirements:
- **Recommended size:** 1200x800 pixels (or 3:2 aspect ratio)
- **Format:** JPG, PNG, or WebP
- **File size:** Keep under 500KB for fast loading

## File Naming:
- `fotobrain.jpg` - For FotoBrain project
- `chatllm.jpg` - For ChatLLM project  
- `lawyerix.jpg` - For Lawyerix project

## How to Add Images:

1. Take screenshots of your projects (or use design mockups)
2. Save them with the names above in this `images/` folder
3. The config.json will automatically reference them
